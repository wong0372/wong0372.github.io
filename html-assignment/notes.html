5 HTML tags I have learnt

<img>
Element in HTML is used to display an image on a webpage. It's a singular tag that requires two essential pieces of information, which are <src> & <alt>
<em>
A string of emphasized, slightly more important text. Screen readers will change their voice for this text.
<a href>
To create a link on a webpage, put what we want to show as the link between these tags, and we use href to say where the link takes us.
<form>
To start adding a form to our HTML page.
<table>
To add a table to the page, all the data to be displayed as part of the table is added inside the opening <table> and closing </table> tags.
