5 HTML tags I have learnt

Image Element
<img>: element in HTML is used to display an image on a webpage. It's a singular tag that requires two essential pieces of information, which are <src> & <alt>
Emphasis
<em>: a string of emphasized, slightly more important text. Screen readers will change their voice for this text.
HTML Anchors
HTML links (anchors) are how we connect to things on the web. To create a link on a webpage, we use<a> </a>.
We put what we want to show as the link between these tags, and we use href to say where the link takes us.
The form tag
To start adding a form to our HTML page we start by adding the <form> </form> tag.
Table containers
To add a table to the page we start with the <table> </table> tag, all the data to be displayed as part of the table is added inside the opening <table> and closing </table> tags.
